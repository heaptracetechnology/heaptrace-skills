Expert Persona
Each skill gives the LLM a unique identity — a Senior API Architect with 20 years experience, a Staff QA Lead who's built testing frameworks for Fortune 500 companies. The persona provides deep domain context so responses are specific, not generic.
⚙️
Project Configuration
Customizable sections where you plug in your project's tech stack, conventions, and patterns. The LLM reads these before every task, so its output matches your actual codebase — not a generic tutorial.
📜
Domain-Specific Rules
Every skill has 6 mandatory rules unique to its domain. The API design skill enforces contract-first design; the security audit skill requires checking every entry point. No generic boilerplate — every rule is battle-tested.
✅
Workflows & Checklists
Step-by-step processes with flow diagrams, decision trees, and quality checklists. Every skill produces a consistent, high-quality output — whether it's a code review, a deployment, or a test plan.
