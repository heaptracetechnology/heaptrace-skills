02

FAQ

Common Questions

The questions we hear most often, answered directly.

What's the minimum engagement size?

Targeted Automation engagements start at 8-week effort. Team Retention starts at 6-month minimum. Hybrid API and Greenfield Build are scoped per engagement.

How fast can you start?

Typical start is 4-6 weeks from contract signing. Faster if scope is well-defined. Slower if compliance review is required (some healthcare and government engagements need 8-10 weeks).

Do you sign BAAs / DPAs?

Yes. We have standard BAA and DPA templates and can review client templates. Standard turnaround on standard contracts is 1-2 weeks.

What about NDA?

Mutual NDA is standard for any technical conversation that goes beyond capability overview. We sign standard mutual NDAs same-day.

Do you do staff augmentation?

No. Heaptrace engagements are outcome-accountable. We're not a body shop. Closest model is Team Retention, but it's still outcome-driven.

Can we use the Skills library without engaging Heaptrace?

Yes. The Heaptrace Skills library is open-source MIT-licensed. Use it free, fork it, modify it. Heaptrace makes money on engagement, not on the library.
