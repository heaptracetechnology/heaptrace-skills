Designer Pack

Design skills — wireframes, user flows, design systems, UX audits, responsive design, micro-interactions, and accessibility.

8 Skills
wireframe
Create detailed ASCII wireframes with layout structure, component placement, responsive breakpoints, grid systems, spacing rhythm, and content hierarchy.
designer / wireframe
user-flow
Map complete user journeys with entry points, decision trees, happy/error paths, exit points, multi-role flows, and state transitions. Use when planning features or debugging UX bottlenecks.
designer / user-flow
design-system
Define or extend a design system with tokens (colors, spacing, typography, shadows, radii), component variants, naming conventions, and documentation.
designer / design-system
ux-audit
Audit existing UI for usability issues using Nielsen's 10 heuristics, consistency checks, cognitive load analysis, task completion paths, and pain point identification.
designer / ux-audit
responsive-design
Plan responsive layouts with mobile-first approach, breakpoint strategy, touch targets, navigation patterns, content reflow, image handling, and typography scaling.
designer / responsive-design
micro-interaction
Design micro-interactions including loading states (skeleton, spinner, progress), transitions, hover/focus feedback, empty states, success/error feedback, and animation specs.
designer / micro-interaction
a11y-design
Design for accessibility following WCAG 2.1 AA standards — color contrast (4.5:1 minimum), focus management, screen reader flow, ARIA landmarks, and keyboard navigation.
designer / a11y-design
design-handoff
Create developer-ready handoff specs with component measurements, design token mapping, interaction states (default/hover/active/disabled/focus), and animation timing.
designer / design-handoff
